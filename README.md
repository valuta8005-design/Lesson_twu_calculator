# Калькулятор семейного бюджета

Небольшой сайт с приветственным лендингом и калькулятором бюджета: считает доходы/расходы, остаток, процент расходов и показывает рекомендацию.

## Возможности

- Автосохранение введённых данных в браузере (localStorage)
- История последних расчётов (до 10)
- Экспорт / импорт данных в JSON
- Переключение темы (светлая / розовая)
- Горячие клавиши: **Enter** — рассчитать, **Esc** — очистить

## Как запустить

Самый простой способ — открыть файл `index.html` двойным кликом.

Если браузер ограничивает некоторые функции при открытии с диска, запустите простой локальный сервер (любой вариант):

### Вариант A: Python

```bash
python -m http.server 5173
```

Затем открыть в браузере `http://localhost:5173/`.

### Вариант B: Node.js

```bash
npx serve .
```

## Структура проекта

- `index.html` — главная страница (лендинг + калькулятор)
- `assets/css/styles.css` — стили
- `assets/js/app.js` — логика калькулятора
- `Калькулятор.html` — исходная «одним файлом» версия (оставлена для истории)
